# Frontend

This directory contains the Next.js 15 frontend application for the VideoConf Platform.

## Structure
- `src/app`: Next.js App Router pages and layouts
- `src/components`: Reusable React components
- `src/lib`: Utility functions, API clients, and AI modules
- `src/context`: React Context providers
- `src/hooks`: Custom React hooks
- `src/services`: Service classes for backend communication
- `src/types`: TypeScript interfaces (if not using shared-types)
- `src/styles`: Global styles and CSS modules
- `public`: Static assets

## Technology Stack
- Next.js 15 (App Router)
- TypeScript
- TailwindCSS
- shadcn/ui
- React Query (for server state)
- Zustand (for client state)
- LiveKit React SDK
- Supabase JS Client
